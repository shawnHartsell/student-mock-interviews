# student mock interviews

various mock interviews for students to practice technical interviews

## Available Interviews

### pokemon-stats/

API integration and data processing interview focusing on working with external APIs, handling JSON data, and implementing search functionality.

### lru-cache-hitrate/

Data structures and algorithms interview implementing an LRU cache with performance tracking. Covers HashMap/LinkedList design, O(1) operations, and adding observability features.

### shopping-cart/

E-commerce cart system interview focusing on object-oriented design, business logic implementation, and money calculations. Tests state management, discount rules, input validation, and edge case handling.
